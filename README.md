# 🏠 House Price Prediction using Linear Regression

## Overview
This project uses Linear Regression to predict house prices based on house size and year. The project includes data visualization, correlation analysis, and an interactive prediction system.

## Features
- Data Exploration and Analysis
- Correlation Heatmap
- Scatter Plot Visualization
- Linear Regression Model
- User Input Prediction System
- House Price Estimation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Workflow
1. Load and clean the dataset
2. Analyze data relationships
3. Generate visualizations
4. Train a Linear Regression model
5. Predict house prices using user inputs

## Visualizations
### Heatmap
Used to identify correlations between variables.

### Scatter Plot
Used to visualize the relationship between house size and price.

## Prediction System
The model accepts:
- House Size
- Year

and predicts the estimated house price.

## Learning Outcomes
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Relationships
- Linear Regression
- Model Prediction
- User Input Handling
